# ADR-001 Project Vision

## Status

Accepted

## Date

2026-06-01

## Context

The goal of AREDVI is to build a production-grade Agentic AI SaaS platform.

The project will serve two purposes:

1. Become a real AI-powered financial intelligence platform.
2. Provide hands-on experience across the full software engineering lifecycle.

## Decision

AREDVI will be built using:

- LangGraph
- FastAPI
- PostgreSQL + pgvector
- Redis
- React
- Docker
- Kubernetes
- AWS

The architecture will prioritize:

- Security
- Scalability
- Observability
- Cost optimization
- Enterprise engineering practices

## Consequences

Development may be slower than typical tutorial projects.

However, the platform will better reflect real-world production systems and provide deeper learning opportunities.
